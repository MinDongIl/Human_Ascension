# Project Human Ascension (인간승급 프로젝트)

> **"지키지 못하면 돈을 내고, 지키면 인간이 된다."**
> 오늘도 침대에 누워 인스타나 처보는 너를 위한 강제 현생 복귀 솔루션.

---

## 등급 및 모욕 시스템 (Tier & Insult)
네 활동량에 따라 매일 아침 네 자존감을 가루로 만들어줄 메시지가 배달된다.

| 등급 | 상태 | AI 메시지 컨셉 |
| :--- | :--- | :--- |
| **LV 1. 폐기물** | 활동 전무 / 차단 사이트 접속 | "네 인생은 필터로도 안 살아나. 부모님 억장 무너지는 소리 안 들리냐?" |
| **LV 2. 그냥 병신** | 최소 활동 (커밋 1개 미만) | "겨우 요거 하고 쉬냐? 뇌에 주름은 있냐? 편의점 알바가 너보다 생산적이다." |
| **LV 3. 노력하는 병신** | 일일 잔디 심기 성공 | "오, 사람 흉내를 내네? 근데 내일이면 다시 병신 될 거 다 안다." |
| **LV 4. 도태 인간** | 이력서 1개 이상 제출 | "자소서 냈냐? 근데 그걸 누가 읽어준대? 그 쓰레기 당장 다시 고쳐." |
| **LV 5. 인간** | 일일 목표 100% 달성 | "기적이다. 오늘 하루는 사람답네. 근데 내일도 가능할까? 절대 못 하지." |

## 핵심 킬러 기능 (Hard Mode)

### 1. [Hard Mode] 현금 페널티 시스템
- **자동 송금:** 당일 목표 미달성 시, 연동된 계좌에서 지정된 벌금이 자동으로 빠져나간다.
- **압박:** "돈 아까우면 코드 짜라." 원초적인 공포를 이용한다.

### 2. [Anti-Distraction] SNS 원천 차단
- **즉시 차단:** 인스타그램, 유튜브 접속 시 즉시 차단 팝업.
- **고수위 모욕:** 네 등급에 맞는 패드립 급 모욕 문구가 전체 화면을 덮는다.

### 3. [Visualized Future] 거지 확률 시각화
- **생존 그래프:** 현재 활동량 기준 1년 뒤 예상 연봉과 '길바닥에서 잘 확률'을 실시간 시각화.

## 기술 스택 (Tech Stack)
- **Backend:** Java (Spring Boot 3.x) - 벌금 결제 API 및 로직
- **Scraper:** Python (FastAPI) - GitHub 및 채용 사이트 크롤링
- **Frontend:** Chrome Extension (Manifest V3) - SNS 차단 UI
- **DB:** PostgreSQL - 유저 로그 및 벌금 트래킹
